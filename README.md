# Automatic Door Lock System using Arduino

## About
A keypad-based automatic door locking mechanism with password
authentication and security lockout feature.

## Technologies Used
- Arduino Uno
- 4x4 Keypad
- Solenoid Lock
- Relay Module
- Embedded C
- Arduino IDE

## Features
- Password-based door access control
- Correct password unlocks the door automatically
- Wrong password 3 times triggers lockout alert
- Buzzer alert on failed attempts

## Circuit Components
- Arduino Uno
- 4x4 matrix keypad
- Solenoid lock
- 5V relay module
- Buzzer
- Jumper wires
